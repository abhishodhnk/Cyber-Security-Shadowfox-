# 🔐 Cyber Security Internship Project – ShadowFox



## 📌 Project Overview

This repository contains the work completed during my **Cyber Security Internship at ShadowFox**.

The project covers both **Beginner and Intermediate level tasks**, focusing on real-world cybersecurity concepts such as:

* Network scanning
* Web enumeration
* Traffic analysis
* Encryption & decryption
* Penetration testing fundamentals

---

## 🌐 Target Website (Beginner Tasks)

http://testphp.vulnweb.com/

*A deliberately vulnerable website used for security testing.*

---

## 🧪 Tasks Performed

### 🔹 Beginner Level

1. **Port Scanning (Nmap)**

   * Identified open ports and services

2. **Directory Enumeration (Gobuster)**

   * Discovered hidden directories

3. **Network Traffic Analysis (Wireshark)**

   * Captured HTTP packets
   * Extracted login credentials

---

### 🔹 Intermediate Level

1. **VeraCrypt Password Cracking**

   * Decoded MD5 hash using CrackStation
   * Accessed encrypted volume

2. **PE Explorer Analysis**

   * Identified executable entry point

3. **Metasploit (Concept Demonstration)**

   * Explored reverse shell payload creation

4. **WiFi Security Analysis (Aircrack-ng)**

   * Studied handshake capture and password cracking
   * Simulated commands due to VM limitations
